1. Class dan Object

Class adalah blueprint untuk membuat object. Object merupakan hasil instansiasi dari class.


class Car {
  String? model;
  String? color;

  void start() {
    print('Car started!');
  }

  void stop() {
    print('Car stopped!');
  }
}

2. Constructor

Constructor digunakan untuk menginisialisasi nilai atribut saat object dibuat.


class Car {
  String? model;
  String? color;

  Car(this.model, this.color);

  void start() {
    print('Car started!');
  }
}

3. Validasi Data

Validasi mencegah kesalahan input dan menjaga konsistensi data.


class Mahasiswa {
  String nama;
  String nim;
  double ipk;

  Mahasiswa(this.nama, this.nim, this.ipk) {
    if (ipk < 0.0 || ipk > 4.0) {
      throw ArgumentError('IPK harus 0.0 - 4.0');
    }
  }
}

4. Encapsulation

Encapsulation menyembunyikan data menggunakan private variable dan getter/setter.


class Person {
  String _name;
  int age;

  Person(this._name, this.age);

  void setName(String name) {
    _name = name;
  }

  String getName() {
    return _name;
  }
}

5. Named Parameter Constructor

Named parameter membuat kode lebih fleksibel dan mudah dibaca.


class Produk {
  final String id;
  final String nama;
  final double harga;
  final int stok;
  final String kategori;

  Produk({
    required this.id,
    required this.nama,
    required this.harga,
    this.stok = 0,
    this.kategori = 'Umum',
  });
}
